What happens if you have colored a lot of pixels and now you want to turn them all off? To get rid of everything at once, type in the function clear(). Type in this:

colorpixel (4, 4, purple)

colorpixel (1, 1, purple)

clear()

colorpixel (2, 2, yellow)

Even though you turned on some purple pixels, the clear() function turns them off right after, and only the yellow one is left! Now put a wait() before and after the clear(), so it looks like this:

colorpixel (4, 4, purple)

colorpixel (1, 1, purple)

wait()

clear()

wait()

colorpixel (2, 2, yellow)

What do you think is going to happen? Run the program and try it out!
